Как правило, для создания программ на C# разработчики используют Visual Studio или какие-то другие IDE типа Rider от компании JetBrains. Но все подобные IDE, хотя и облегчают разработку, в то же время скрывают множество мельчайших подробностей по созданию программ. И в реальности мы даже можем обойтись без той же Visual Studio, используя только инфраструктуру .NET Core CLI. Рассмотрим, как использовать .NET Core CLI и компилировать приложение из командной строки. Поэтому создадим на диске C какую-нибудь новую папку для проекта. Назовем ее, к примеру, helloapp. Откроем командную строку и с помощью команды cd перейдем к этой папке.
